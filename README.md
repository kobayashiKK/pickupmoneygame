# pickupmoneygame ☂💴

空からお金が降ってくるのを、**逆さまにした傘**で拾うブラウザゲームです。
インストール不要。`index.html` をブラウザで開くだけで遊べます。

## 遊び方

1. `index.html` をダブルクリック（またはブラウザにドラッグ）して開く
2. 「スタート」を押す
3. 左右に動いて、傘で落ちてくるお金をキャッチ！

## ルール

- 🪙 **小銭**（1〜500円）と 💵 **紙幣**（1,000〜10,000円）が降ってきます
- 拾った金額がスコアになります
- 💣 **爆弾**を拾うとライフが1つ減ります（失敗要素）
- ライフは3つ。爆弾を3回拾うか、制限時間60秒で **ゲームオーバー**
- 時間が経つほどスピードが速くなり、爆弾も増えていきます
- **ハイスコア**はブラウザに自動保存されます（localStorage）

## 操作

| 操作 | 方法 |
| --- | --- |
| 移動 | `←` `→` キー / `A` `D` キー |
| 移動（タッチ・マウス） | 画面をドラッグ |
| 開始・リトライ | スタートボタン / `Space` / `Enter` |

## 技術

- 依存ライブラリなしの素の HTML + Canvas + JavaScript 1ファイル構成
- PC・スマホ両対応（レスポンシブ＆タッチ操作）
