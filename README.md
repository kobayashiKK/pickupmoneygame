# pickupmoneygame ☂💴

空からお金が降ってくるのを、**逆さまにした傘**で拾うブラウザゲームです。
インストール不要。`index.html` をブラウザで開くだけで遊べます。

## 遊び方

1. `index.html` をダブルクリック（またはブラウザにドラッグ）して開く
2. 「スタート」を押す
3. 左右に動いて、傘で落ちてくるお金をキャッチ！

## ルール

- 🪙 **小銭**（1〜500円）と 💵 **紙幣**（1,000〜10,000円）が降ってきます
- 拾った金額がスコアになります
- 💣 **爆弾**を拾うとライフが1つ減ります（失敗要素）。ハートが減って画面が揺れます
- ライフは3つ。**爆弾を3回拾うとゲームオーバー**（制限時間なし。どこまで稼げるかに挑戦！）
- 時間が経つ・得点が伸びると **レベルアップ**。難易度の上がり方は **「傘が小さくなる」と「お金が速くなる」が交互** に発生します（傘が最小になった後は速度アップのみ）
- いつでも **一時停止（中断）** できます。再開・最初から・タイトルへ戻るが選べます
- ゲーム終了時に、稼いだ金額に応じた **「〇〇レベル」と買えるもの** を表示します
- **ハイスコア**はブラウザに自動保存されます（localStorage）

## 操作

| 操作 | 方法 |
| --- | --- |
| 移動 | `←` `→` キー / `A` `D` キー |
| 移動（タッチ・マウス） | 画面をドラッグ |
| 開始・リトライ | スタートボタン / `Space` / `Enter` |
| 一時停止・再開 | 画面上部の ⏸ ボタン / `P` / `Esc` |

## 技術

- 依存ライブラリなしの素の HTML + Canvas + JavaScript 1ファイル構成
- PC・スマホ両対応（レスポンシブ＆タッチ操作）
